Update logs

This page is for important update logs for my game "Tower Defense Game Extension".
Information for some updates may not be available, so if there are any missing stuffs about the game, you can request to add it into this page!
I do not receive requests for adding update logs for versions older than 1.6.0.

TDG 1.8.0 (release date 30 March - 3 April) Beta release

I'm currently working on another update for the game. Along with a bunch of bugfixes, there are also a ton of changes made to the game.
This version adds a lot of new stuffs not mentioned before, such as:

  • New Earth turret (see here for information).
  • Mob cap for normal & boss wave added, for reducing time between waves.
  • Regen boss added (see here for information).
Changes were made to the game, such as:
  • Turret costs re-balanced (some got cheaper, while others got more expensive).
  • Special upgardes for Fire turret have become stronger.
  • Effect duration of elemental turrets adjusted.
