TDG 1.8.0 (latest) Beta release

This is the latest beta version for TDG 1.8.0.
A lot of bugs have been fixed, and a lot of features have been added!
Notable changes include:

  • Special upgrades for Fire turret now work.
  • More wave variations for more interesting games.

TDG 1.6.2 Minor release

An older version of the game, mainly an update for TDG 1.6.0.
This version does not support turret selling and didn't have Forest maps, so it was kinda empty.

TDG 1.6.2 (fps option test) Beta release

The second beta for TDG 1.6.0 (the last being 1.6.3, but we won't talk about that now)
This version added options for switching between 30fps, 60fps and 120fps. Note that high framerates are unstable, so be aware!
Upgrades don't work for some reason, but you should be able to play it still.

TDG 1.3.0 Major release

This update is pretty old, so don't expect this to have many features yet.
The update added Gear and Electric turrets, which are still in the game to this day.
Gear turret was broken after 1.6.0 (where upgrades were added), but the bug was fixed after the latest 1.8.0 beta came out
